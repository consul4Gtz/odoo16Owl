Product Package on POS with Pricelist Odoo Apps

POS Product Package Based Pricelist odoo app helps users to create, sell and manage point of sale products in multiple packages for the same product with different quantity and different price options using pricelist. Users can set different prices for various packages of the same product using the same pricelist. Users can select available product packages of products at the point of sale and also enter the quantity of packages and can see them in POS receipt. User also have option to enable validation to select customer first before selectiong pacage product.

Features

Manage Product Packages with using Pricelist

User can manage price for the package of a product by using the pricelist.

Package Product Customer Validation

User can configure to add validation to select customer first before selection package product.

Available Package Options for a Product in POS

User can see available product packages options in POS screen.

POS Receipt Show Product Packages

In point of sale receipt user can see product packages name with price.
Demo Request Video Tutorial

Product Packagings

Under Inventory > Configuration > Product Packagings menu user can create multiple packages for same product with different quantity.

Available Package for Product

User can see available packages for product.

Apply Pricelist on Product Package

User can select product and available packages for that product and set price computation.

Point of Sale Package Product

User can see product with "Package product" label on point of sale screen.

On clicking package product new popup will open, Here user can see all available packages for product and can enter quantity for package and view total price of package.

Package Product on Order Receipt

User can see point of sale product package name with quantity in point of sale receipt.

Package Product Customer Configuration

If enable "Customer Is Required" option then before selecting package product customer must selected, Other wise warning will raise.

Package Product Warning

If customer not selected then warning will raise.

User can see product with "By Lots" traceability.

Package Product Option

If user select a package product having traceability by lot number then popup will show, Here user can select option to apply lot/serial number for product or select package product.

If User select "Apply a Lot/Serial Number(s) / Product Configurator" option then default lot/serial number popup will show, and product added to cart.

If user select "Add Package Product" option then user can see all available packages for product.

Selected product pack added to POS cart.
